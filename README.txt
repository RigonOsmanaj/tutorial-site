"This is a simple one-page website built using HTML, SCSS, and Bootstrap. It showcases basic features such as a carousel, responsive cards, and a clean layout, designed to test and demonstrate web development techniques.

Features
- Responsive design with Bootstrap.
- Carousel showcasing top destinations.
- Uniform and visually appealing cards.

Purpose
This project serves as a playground for experimenting with HTML, SCSS, and Bootstrap to implement responsive and interactive web components."
